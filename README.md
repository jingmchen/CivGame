# CivGame
A game I am making with my CS friend in Godot 4.6.3 Mono C# .NET10.0 to be released on Steam

I really like games. But the games I want, can only be created by me.

More updates later. Will not be open sourced to be sold on Steam

Inspired by Rimworld, rather than hardcoding each character, terrain, building stats into code, we use an architecture to load them iteratively from Assets folder using defs XML. (All stats and data to be in XML format strictly). Engine loads them in by iterating XML defs in Assets folder and Sprites in Assets folder
